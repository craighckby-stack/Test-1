# AGI-KERNAL: Cumulative Memory Core
AGI-KERNAL is a cumulative memory core for distributed AI systems.
It utilizes a combination of blockchain, AI and federated learning.

## Features:
* Utilizes Polygon blockchain
* Employs LLaMA-3.3-70b Transformer model
* LLaMA Tokenizer
* Decentralized Federated Learning with 512 peers
* Global Model Sharing
* Mass-loss guard to prevent code truncation

## Roadmap:
* Improve code efficiency and stability
* Enhance security measures
* Expand to other blockchain networks
* Implement more advanced AI models
